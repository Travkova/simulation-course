## Метод конечных разностей для уравнения теплопроводности

**Задание:**  
Реализовать моделирование изменения температуры в пластине на основе одномерного уравнения теплопроводности с использованием метода конечных разностей.
Выполнить моделирование с различными шагами по времени и по пространству.  
Заполнить таблицу значений температуры в центральной точке пластины после 2 секунд модельного времени.
### Одномерное уравнение теплопроводности:
$$\rho c \frac{\partial T}{\partial t} = \lambda \frac{\partial^2 T}{\partial x^2}$$

### Суть метода:
Основная идея метода заключается в замене непрерывной области, в которой ищется решение дифференциального уравнения, на дискретную сетку узлов. Производные заменяются конечно-разностными отношениями (разностями):

$$\rho c \frac{T_i^{n+1} - T_i^n}{\Delta t} = \lambda \frac{T_{i-1}^{n+1} - 2T_i^{n+1} + T_{i+1}^{n+1}}{h^2}$$

- $$T_i^n\$$ - температура в узле i в текущий момент времени
- $$T_i^{n+1}\$$ - температура в узле i в следующий момент времени
- $$h\$$ - шаг по пространству
- $$\Delta t\$$ - шаг по времени

В результате исходное сложное дифференциальное уравнение превращается в систему алгебраических уравнений:
$$A T_{i-1}^{n+1} - B T_{i}^{n+1} + C T_{i+1}^{n+1} = F_i$$

Коэффициенты представляются в виде:
- $$A = \frac{\lambda}{h^2}$$
- $$B = \frac{2\lambda}{h^2} + \frac{\rho c}{\Delta t}$$
- $$C = \frac{\lambda}{h^2}$$
- $$F_i = -\frac{\rho c}{\Delta t} T_i^n$$

Данная система решается **методом прогонки**:
- **Прямой ход**: Информацию о левом граничном условии переносим на всю систему.
$$\alpha_i = \frac{A}{B - C \alpha_{i-1}}, \quad \beta_i = \frac{C \beta_{i-1} - F_i}{B - C \alpha_{i-1}}$$

- **Обратный ход**: Последовательно находим значения температур во всех узлах сетки, отталкиваясь от известного правого граничного условия.
$$T_i^{n+1} = \alpha_i T_{i+1}^{n+1} + \beta_i$$
