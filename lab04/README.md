### Базовый датчик случайных чисел

**Задание:**
- реализовать базовый датчик случайных чисел;
- вычислить выборочные среднее и дисперсию:
  - для реализованного датчика;
  - для встроенного генератора языка программирования;
- размер выборки — **100 000 значений**;
- сравнить результаты с теоретическими;
- сделать вывод.

**Решение:**
  Базовый датчик случайных чисел реализуется с помощью Мультипликативного конгруэнтного генератора. Данный алгоритм генерации псевдослучайных чисел основан на линейной рекуррентной формуле: $$X_{n+1} = (beta \cdot X_n) \pmod M$$, где:
  
  - $X_{n+1}$ - следующее псевдослучайное число
  - $X_n$ - текущее
  - $beta$ - мультипликативный множитель
  - $M$ - модуль
  - $X_0$ - начальное значение(seed)
Если нужно получить дробное число(например, в диапазонре от 0 до 1), результат делится на модуль: $$result = X_n / M$$

В данной работе использовались значения: 
- $X_0$ = 42
- $M$ =$2^{63}$
- $beta$ =$2^{32} + 3$
