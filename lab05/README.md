**Теория для реализации приложения да/нет:**

Вероятность события можно представить геометрически как отрезок на интервале [0, 1]:

$$P\{\text{random} < p\} = p$$, где random - СВ, равномерно распределённая на интервале [0, 1], p - вероятность наступления события

Алгоритм: 
1. A := α (генерация СВ)
2. Если α < p, то событие наступило (ДА). Иначе: событие не наступило (НЕТ)

**Теория для реализации приложения шара предсказаний:**

События $A_1, A_2, ..., A_m$ образуют полную группу попарно несовместных событий, если:

$$\sum_{i=1}^{m} p_i = 1$$, где $p_i$ = P{Aᵢ} — вероятность наступления события $A_i$.

Алгоритм генерации события из группы событий:

1. A := α (генерация СВ); k := 1 (Счётчик, индекс текущего события)
2. A := A - pₖ
3. Если A ≤ 0 → произошло событие Aₖ
4. Иначе k := k + 1, повторить шаг 2
