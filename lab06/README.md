### Имитационное моделирование дискретных случайных величин

**Задание:**
- реализовать генерацию дискретной случайной величины, заданной рядом распределения;
	- вычислить эмпирические вероятности;
	- вычислить выборочные среднее и дисперсию, их относительные погрешности;
	- вычислить статистику χ² и применить критерий χ² при:
	  - N = 10, 100, 1 000, 10 000;
- Реализовать генератор нормальной случайной величины, 
	- построить гистограммы 
	- сравнить точность моделирования при разных объёмах выборки.
- сделать вывод.

### Используемые методы

Для генерации дискретной СВ был использован метод обратной функции распределения:
- Вычисляется массив накопленных(кумулятивных) вероятностей
- Генерируется равномерное случайное число $r \sim U(0, 1)$
- Выбирается значение, соответствующее интервалу накопленных вероятностей

Для генерации нормальной СВ был использован метод Бокса — Мюллера:
- Генерируются две независимые случайные величины $U₁, U₂ \sim U(0, 1)$
- Вычисляется: $$Z = \sqrt{-2 \ln U_1} \cdot \cos(2\pi U_2)$$
- Результат: $$X = Z \cdot \sigma + \mu$$

### Выводы

По результатам моделирования *дискретной СВ* можно сделать вывод, что с ростом объёма выборки эмпирические вероятности, среднее и дисперсия сходятся к теоретическим значениям, что подтверждает закон больших чисел. Критерий хи-квадрат при N ≥ 1000 подтверждает соответствие распределения заданному.

По результатам моделирования *нормальной СВ* можно сделать вывод, что с увеличением N выборочные характеристики приближаются к заданным параметрам μ и σ. Иногда при объёмах выборки 10 и 100 критерий Хи-квадрат может не выполняться.
