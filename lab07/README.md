### Марковская модель погоды

**Задание:**  
Смоделировать погоду по дням:

- 1 — ясно  
- 2 — облачно  
- 3 — пасмурно  

Единица времени — **1 день**.  
Задать интенсивности переходов между состояниями.

**Требования:**
1. Выполнить моделирование в «реальном» времени с визуализацией.
2. Провести статистическую обработку результатов.
3. Сравнить эмпирическое распределение с теоретическим стационарным.

*Ключевые алгоритмы*

1. Задаётся инфинитезимальная матрица $Q$, описывающая скорости переходов между состояниями. Сумма элементов каждой строки должна равняться нулю.
2. Время пребывания в состоянии $\tau$, которое имеет экспоненциальное распределение с параметром $\λ_i = -\q_{ii}$ (интенсивность выхода).
3. Выбор следующего состояния next_state_idx: выбирается как ДСВ с вероятностями $p_{ij} = -\frac{q_{ij}}{q_{ii}}$, если j не равно i. Если $\j=\i$  $p_{ij} = 0$.
4. Вычисление стационарного распределения: Система уравнений Колмогорова $\pi Q = 0$ при условии нормировки $\sum \pi_i = 1$; Решаем методом Гаусса через np.linalg.solve()
5. Статистическая обработка: эмпирические вероятности считаются как суммарное время, проведённое в состоянии, разделить на общее время моделирования.
