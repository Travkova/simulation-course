### Марковская модель погоды

**Задание:**  
Смоделировать погоду по дням:

- 1 — ясно  
- 2 — облачно  
- 3 — пасмурно  

Единица времени — **1 день**.  
Задать интенсивности переходов между состояниями.

**Требования:**
1. Выполнить моделирование в «реальном» времени с визуализацией.
2. Провести статистическую обработку результатов.
3. Сравнить эмпирическое распределение с теоретическим стационарным.

*Ключевые алгоритмы*
1 Генерация экспоненциального распределения tau: Время пребывания в состоянии Марковского процесса с непрерывным временем имеет экспоненциальное распределение с параметром λ = интенсивность выхода.
2 Выбор следующего состояния next_state_idx: Используется метод обратного преобразования для дискретного распределения.
3 Вычисление стационарного распределения: Система уравнений πQ = 0 (транспонированная: Qᵀπᵀ = 0); Добавляем уравнение нормировки Σπᵢ = 1; Решаем методом Гаусса через np.linalg.solve()

*Вывод:* Реализована программа моделирования погоды как марковский процесс с непрерывным временем, заданный с помощью матрицы интенсивностей переходов Q. Статистика сохраняется в таблицу weather_statistics.csv
