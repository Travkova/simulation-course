### Система массового обслуживания M/M/1

Моделирование простейшей системы M/M/1.  

### Отчёт по проделанной работе

Система М/М/1 характеризуется:
- Входящий поток: является пуассоновским с интенсивностью $\lambda$. Интервалы между поступлениями заявок имеют экспоненциальное распределение.
- Время обслуживания: имеет экспоненциальное распределение с интенсивностью $\mu$.
- Кол-во каналов: 1
- Дисциплина очереди: FIFO (Первый пришёл, первый обслужен).
- Условие стабильности: $\rho$ = $\lambda$ / $\mu$ < $1$, иначе очередь растёт до бесконечности.

**Ключевые аналитические формулы:**
| Показатель | Формула |
|------------|---------|
| Загрузка системы | `ρ = λ / μ` |
| Среднее число клиентов в системе | `L = ρ / (1 - ρ)` |
| Среднее время ожидания в очереди | `Wq = ρ / (μ·(1 - ρ))` |
| Вероятность простоя (свободный оператор) | `P₀ = 1 - ρ` |
| Стационарное распределение числа клиентов | `P(L = n) = (1 - ρ)·ρⁿ` |

**Выводы**: При параметрах $\lambda = 2$, $\mu = 2.5$  , $T = 1000$ (общая длительность моделирования) получаются следующие результаты:
1. Форма распределений верная:
   - Распределение числа клиентов имеет характерную геометрическую форму (убывающая кривая);
   - Распределение времени ожидания показывает экспоненциальный "хвост"
   - Оба графика соответствуют теории для M/M/1
2. Система стабильна: $\rho=0.8 < 1$, очередь не растёт до бесконечности
3. Эмпирическая вероятность простоя $P_0$ завышена (0.239; теоретическая 0.2), а время ожидания занижено (1.09 вместо 1.6). При Т > 1000 эмпирические характеристики лучше сходятся к теоретическим.
